# OpenAI Flask API

Simple Flask API to interact with OpenAI.

## Installation

pip install -r requirements.txt

## Run

python API.py

## Endpoint

POST /perguntar

### Request

{
  "texto": "Como criar um robô Padrão RPA?"
}

### Response

{
  "### Pré-requisitos
1. **Python** instalado (preferencialmente Python 3).
2. **Selenium** instalado. Você pode instalar o Selenium com o comando:
   ```
   pip install selenium
   ```
3. **WebDriver** do navegador que você está utilizando. Por exemplo, se você estiver usando o Chrome, precisará do ChromeDriver...."
}
